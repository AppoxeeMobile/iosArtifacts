Appoxee SDK - US
===========
This repository contains the Appoxee iOS SDK.

Integration
-----------

*Cocoa pods

```
target 'project_name' do
    pod 'AppoxeeSDK'
end
```

*OR*


```
target 'project_name' do
    pod 'AppoxeeSDK', :git => 'https://github.com/AppoxeeMobile/iosArtifacts.git', :tag => '4.0.10'
end
```

*Manual

Drag and drop the .framework to your project.
